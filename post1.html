Making a slide show in HTML

Photography is something I really enjoy doing so I thought it would be fun to look into different ways to display pictures in HTML. After looking up various formats, I decided to explore how to create a slideshow of pictures using HTML, Javascript and CSS. I based my implementation off of a w3schools article, which can be found here. Through this experience I learned how to format images of different sizes in the same amount of space without distorting them, how to write a javascript function to loop through and display different div elements and how to create working buttons without using the button tag.
