Making A Slideshow In HTML

Photography is something I really enjoy doing so I thought it would be fun to look into different ways to display pictures in HTML. After looking up various formats, I decided to explore how to create a slideshow of pictures using HTML, Javascript and CSS. I based my implementation off of a w3schools article, which can be found here. From this tutorial I not only learned how to create a slideshow of pictures but I also learned how to write a javascript function to loop through and display different div elements and how to create working buttons without using the button tag. Below is the javascript code contained within the script tag. This code makes it possible for only one photo to be displayed at a time and for the arrow buttons to successfully switch to the next photo. 

 
<script>

var slideIndex = 1;
showSlides(slideIndex);

function plusSlides(n) {
  showSlides(slideIndex += n);
}

function currentSlide(n) {
  showSlides(slideIndex = n);
}

function showSlides(n) {
  var i;
  var slides = document.getElementsByClassName("mySlides");

  if (n > slides.length) {slideIndex = 1}    
  if (n < 1) {slideIndex = slides.length}
  for (i = 0; i < slides.length; i++) {
      slides[i].style.display = "none";  
  }
  slides[slideIndex-1].style.display = "block";  
}
</script>

The showSlides function accesses all the div elements that are in the mySlides class which contains all the elements that have an image. The function then loops through all those div elements and sets their display to none so they are not visible. Finally, it sets the photo at index n to be visible by setting the display to block. I thought that toggling the display property for each div element was a really nice way to make a single image visable. This approach is much easier to understand and implement as opossed to trying to replace or create a new image every time.

The previous and next arrow elements have onclick attributes which call the plusSlides function as seen below. 

 
<a class="prev" onclick="plusSlides(-1)"> ❮ </a>

I've never used the onclick attribute before so this was a nice opportunity to see how it works. Adding onclick to an element is a very easy and simple way to add functionality. In the case of the slideshow the onclick attribute allowed the arrow symbols to become buttons that once clicked call the plusSlides function. 

 
function plusSlides(n) {
  showSlides(slideIndex += n);
}

The plusSlides function adds one to the slideIndex and calls the showSlides function to display the image. Something I found really cool was the ability to use the same plusSlides function for both the previous and next button. To pass a 1 or -1 to the function was a really simple and smart way to reuse code and keep things clean.

The one major challenge I ran into when implementing the picture slideshow was the formatting of the previous and next arrow buttons. I found that if I added images with different dimensions then the arrows would end up moving to fit the different sized images. In order to fix this I added a height and object-fit property to the images. 

  img {
    vertical-align: middle;
    width: 100%;
    object-fit: contain;
    height: 70vh;
}

Overall, I really enjoyed learning how to create this slideshow. I think that this could have a lot of pratical and creative uses in a webpage. It can also be expaned upon to include things such as numbers to show how many pictures there are or thumbnails with the other images.
