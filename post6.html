The Lorem ipsum text under each section starts off with a display of none. Then using javascript once one of the sections is clicked the text’s display changes to block. The javascript used is below. 

 
      for (i = 0; i < coll.length; i++) {
        coll[i].addEventListener("click", function() {
          this.classList.toggle("active");
          var content = this.nextElementSibling;
          if (content.style.display === "block") {
            content.style.display = "none";
          } else {
            content.style.display = "block";
          }
        });
      }

I have seen most of these javascript statements before except for the ‘this.nextElementSibling’ statement. After looking into it and reading this article, I learned that it will access the next element at the same level as the element it is being called on. This allows only the text that is associated with Open Section 1 to be displayed once Open Section 1 is clicked. This is extremely useful for times when you have multiple elements in a class but only want to access certain ones. It also allows you to have associations between elements.
